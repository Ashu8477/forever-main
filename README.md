# вҡҷпёҸ Installation

Clone Repository

```bash
git clone https://github.com/Ashu8477/forever-main.git
```

Go to Project

```bash
cd forever-main
```

Install Dependencies

### Frontend

```bash
cd frontend
npm install
```

### Backend

```bash
cd backend
npm install
```

### Admin

```bash
cd admin
npm install
```

---

# в–¶пёҸ Run Project

Frontend

```bash
cd frontend
npm run dev
```

Backend

```bash
cd backend
npm run server
```

or

```bash
npm start
```

Admin

```bash
cd admin
npm run dev
```
